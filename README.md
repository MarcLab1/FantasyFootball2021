# Fantasy Football 2021

A collaboration of useful information I found from r/fantasyfootball thrown together entirely to demonstrate some Android concepts.

## Concepts 

  * Kotlin
  * Room
  * Coroutines 
  * ViewPager2
  * Tablayout
  * RecyclerView
  * GSON - used to convert objects into JSON
